# Практическая работа №3 — Zephyr IoT Security

## Порядок выполнения

1. `1_install_deps.sh` — установка зависимостей
2. `2_setup_venv.sh` — создание Python окружения (активировать перед каждой сессией)
3. `3_clone_zephyr.sh` — загрузка исходного кода Zephyr
4. `4_install_sdk.sh` — установка SDK и инструментов анализа трафика
5. `5_build_server.sh` — сборка echo-сервера
6. `6_run_server.sh` — запуск сервера (Терминал 1), запомнить PORT
7. `7_sniff_traffic.sh PORT` — перехват трафика (Терминал 2)
8. `8_connect_client.sh PORT` — подключение клиента (Терминал 3)

## Что произойдёт
Всё что введёшь в терминале 3 — будет видно в терминале 2 в открытом виде.
